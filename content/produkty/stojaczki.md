# Stojaczki (do proporczyków i flagietek)

## Purpose

Detail page for small desktop stands used to display flagietki and proporczyki — typically a desktop or shelf-sized accessory.

## Verbatim content from current site

### Page title
STOJACZKI - J-ART Flagi Osielsko

### Section heading
**STOJACZKI do proporczyków i flagietek**

### Description

> Stojaczki — różnorodne typy podstawek: z maszcikami jedno- i wieloramiennymi, metalowe złote lub srebrne ze zwieńczeniami, oraz różnego rodzaju podstawy metalowe złote, srebrne, kamienne, płaskie, sferyczne. Stojaczki mogą być także całkowicie drewniane.

## Suggested UI sections for the redesigned page

1. **Page hero**
   - H1: "Stojaczki do proporczyków i flagietek"
   - Lead: "Eleganckie podstawki na biurko lub półkę — metalowe, kamienne, drewniane. Jedno- lub wieloramienne."

2. **Typy stojaczków** (grid)

   **Wg ramion (maszcików):**
   - **Jednoramienne** — pojedyncza flagietka lub proporczyk
   - **Wieloramienne** — kilka flag w jednym stojaku (idealny do pokoju konferencyjnego z flagami państw)

   **Wg materiału i wykończenia:**
   - **Metalowe złote** — reprezentacyjne, klasyczne
   - **Metalowe srebrne** — eleganckie, nowoczesne
   - **Kamienne** — stabilne, premium
   - **Całkowicie drewniane** — naturalne, ciepłe
   - **Ze zwieńczeniami ozdobnymi** — na szczycie maszcika

   **Wg kształtu podstawy:**
   - Płaska
   - Sferyczna (kulista)

3. **Zastosowania** (4-card grid)
   - **Biurko prezesa** — pojedyncza flaga państwowa lub firmowa
   - **Sala konferencyjna** — flagi państw uczestników spotkania
   - **Recepcja** — flagi firmowe i partnerskie
   - **Prezent korporacyjny** — eleganckie zestawy proporczyków

4. **Galeria typów** (visual catalog)
   - Request photo per type from client

5. **Cross-sell**
   - "Potrzebujesz flagietek?" → `/oferta/flagietki`
   - "Lub proporczyków?" → `/oferta/proporczyki`

6. **CTA**
   - "Skompletujemy zestaw stojaczków z Twoimi flagami"
   - Button: "Zapytaj o wycenę" → `/kontakt`
