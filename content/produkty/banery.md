# Banery

## Purpose

Detail page for advertising banners (PVC-coated frontlit and mesh).

## Verbatim content from current site

### Page title
BANERY - J-ART Flagi Osielsko

### Section heading
**BANERY**

### Description

> Banery — wodoodporny nadruk na tkaninie powlekanej tworzywem PCV — powierzchnia lita frontlit 500 g/m² lub siatkowa 330 g/m² (mesh).
>
> Standardowo brzegi podwójnie zagrzewane + kółka plandekowe co ok. 50 cm. Grafika dowolna.

## Suggested UI sections for the redesigned page

1. **Page hero**
   - H1: "Banery reklamowe"
   - Lead: "Wodoodporne banery PCV — frontlit lita 500 g/m² lub mesh siatkowy 330 g/m². Dowolny rozmiar, dowolna grafika."
   - Hero image: `BANER1.jpg` or `kladka.png`

2. **Dwa rodzaje banerów** (comparison cards)

   | | Frontlit | Mesh |
   |---|---|---|
   | **Materiał** | Tkanina lita powlekana PCV | Tkanina siatkowa powlekana PCV |
   | **Gramatura** | 500 g/m² | 330 g/m² |
   | **Przepuszczalność wiatru** | Nieprzepuszczalny | Przepuszcza wiatr (mesh) |

   | **Czytelność z bliska** | Pełna kolorystyka | Lekko przesłonięta strukturą |

3. **Specifikacja wykończenia**
   - Brzegi podwójnie zagrzewane (trwałe, nie strzępią się)
   - Kółka plandekowe co ok. 50 cm — gotowe do mocowania
   - Wodoodporne — można używać na zewnątrz w każdą pogodę

4. **Zastosowania** (4-card grid)

   - **Sport** — banery przy boiskach, w halach
   - **Promocja sklepów** — wystawy, witryny, parkingi
